## 方案讨论

| # | 问题 | 回答 | 影响 |
|---|------|------|------|
| 1 | 接口参数名大小写？ | `UserName` / `Password` | LoginRequest 字段改为大写开头 |
| 2 | 登录后跳转？ | 创建首页，跳转首页 | 需新建 home 页面和路由 |
| 3 | LoginResponse 字段？ | `{ Token, UserName, ... }`，只保存 Token | LoginResponse 类型调整，只持久化 Token |
| 4 | Token 如何传给后续接口？ | 请求头 `Token: xxx` | 拦截器从 `Authorization: Bearer` 改为 `Token` |

## 总体思路

1. 登录表单提交时，Password 经 `hex_sha1` 加密，调用 `apiAuthLogin({ UserName, Password })` 接口
2. 登录成功后将 `Token` 存入 `useUserStore` 并持久化到 localStorage
3. 全局请求拦截器改为 `Token: xxx` 传给后续接口
4. 登录成功后跳转首页；创建首页占位页面和路由
5. loading 状态提升到父组件，login-form 改为受控
