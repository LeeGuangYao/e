# 需求背景

重构当前机票城市选择功能，优化城市输入与选择体验，整体UI风格现代化、简洁、高级，提升用户在PC端/H5端的操作效率与视觉体验。

---

# 接口说明

## 1. 获取城市列表接口

接口地址：

api/flight/getFlightCityList

请求方式：

GET

请求参数：

无

用途：

用于初始化热门城市、城市分组列表展示。

---

## 2. 城市模糊查询接口

接口地址：

api/flight/searchAirportCity/v2

请求方式：

POST

请求参数：

```json
{
  "CountryType": 1,
  "KeyWord": "da"
}
```
